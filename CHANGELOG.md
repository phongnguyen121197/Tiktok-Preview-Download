# TikTok Preview & Download - Changelog

## [2.0.0] - 2026-01-16

### Added
- **Monitor Page**: New dashboard to show FastMoss session status (placeholder for future multi-user feature)
- **About Page**: App info, donate section, system information
- Updated Sidebar with Monitor and About menu items
- Version badge in sidebar footer showing "v2.0.0 Ready"

### Note
This version is based on v1.2.0 with additional UI components.
Multi-user FastMoss session pooling will be implemented in a future update.

## [1.2.0] - 2024-01-14

### Features
- **Video Player hoàn chỉnh**
  - Controls luôn hiển thị
  - Click video để play/pause
  - Thanh tua video
  - Hiển thị thời gian current/total
  
- **FastMoss - Single Window Mode**
  - Tất cả navigation trong 1 cửa sổ
  - Shared Session partition
  - Refresh Session API
  
- **Shop Product Detection**
  - Tự động phát hiện sản phẩm gắn trong video
  - Hiển thị Product title, Shop info

### Fixed
- Fixed video download with redirects
- Improved error handling
